Project 1

Project 1: Soil Erosion Awareness Website

Description:

This project involved designing and developing a comprehensive website focused on raising awareness about soil erosion. The website educates users on the causes, effects, and prevention strategies related to soil erosion, with interactive elements such as quizzes, infographics, and real-world case studies to make the information engaging and accessible.

Key Features:

Informative Content: Includes detailed sections on types of soil erosion, environmental impact, and ways to mitigate damage. Interactive Quiz: An educational quiz at the end of each section to reinforce learning. Responsive Design: Built using HTML, CSS, and JavaScript, the website is fully responsive, making it accessible on both mobile and desktop devices.

Technologies Used:

HTML, CSS, JavaScript for front-end development Figma for designing the layout and user interface.
